# Expiry Check

Expiry Check is a SaaS inventory and expiration date management platform.

Goals:

- Track expiry dates
- Reduce product waste
- Reduce financial losses
- Inventory monitoring
- Smart alerts
- Analytics
- Future AI demand forecasting

Target Users:

- Supermarkets
- Retail stores
- Warehouses
- Pharmacies
